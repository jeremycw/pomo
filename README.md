# Usage
```
Usage: pomo [options]
    -h, --help                       Display this message
    -w, --work-minutes N             N minutes for each pomodoro (default 25)
    -s, --break-minutes N            N minutes for standard breaks (default 5)
    -b, --long-break-minutes N       N minutes for long breaks (default 15)
    -i, --long-break-interval N      Complete N pomodoros to get a long break (default 4)
    -l, --logfile FILE               Path to logfile
```

# Dependencies
- Ruby 1.9.3+
- terminal-notifier
